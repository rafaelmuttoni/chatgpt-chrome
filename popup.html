Search any app from Chrome's search bar

  1. To get started, open Chrome's settings by clicking the three dots at top-right.
  2. Click "Search engine" then "Manage search engines", or go directly to chrome://settings/searchEngines.
  3. From here, you can add new search engines.
  4. On the modal, add a name (ex.: ChatGPT).
  5. Add a shortcut (ex.: gpt) which is what you're going to type on the tab bar to activate the search.
  6. And the URL: https://chat.openai.com/?text=%s or https://chat.openai.com/?model=gpt-4&text=%s if you want to use GPT 4.

For more detailed instructions, please see the official Google support article.
